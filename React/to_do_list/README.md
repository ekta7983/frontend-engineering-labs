# React Todo App

A simple and interactive Todo Application built using React.  
This project helped me practice core React concepts like state management, event handling, conditional rendering, and localStorage persistence.

---

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit existing tasks
- Persistent storage using localStorage
- Dynamic UI updates using React state

---

## Concepts Practiced

This project helped in understanding:

- React functional components
- useState hook
- useEffect hook
- Controlled components
- Conditional rendering
- Array methods in React:
  - map()
  - filter()
- Immutable state updates
- localStorage integration
- Event handling
